- 原理
  - 瀏覽器端向伺服器發起請求，伺服器找到檔案之後傳送過去。
  - 使用後端程式才會有執行的動作，執行完銜接到網頁
  - 瀏覽器只有拿到頁面原始碼，由各自瀏覽器執行。
    - 在各個瀏覽器下執行結果會不一樣
  - 頁面原始碼
    - Html
    - CSS
    - JavaScript
  - 瀏覽器與伺服器之間是無狀態模式 ( http )
  - 每一個頁面都是獨立執行
